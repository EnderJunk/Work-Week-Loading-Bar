# Work Week Loading Bar

A simple web app that shows your progress through the work week with a real-time loading bar. Track how close you are to the weekend!

## Live Demo

ðŸ”— [View Live Site](https://work-week-loading-bar.onrender.com)

## Features

- Select your working days
- Set custom shift times
- Real-time progress bar
- Auto-saves your schedule
- Updates every minute

## How to Use

1. Select your working days
2. Set your shift start and end times
3. Click "Save Schedule"
4. Watch your progress update automatically

## Technologies

- HTML, CSS, JavaScript (ES6 Modules)
- Local Storage for saving schedules

## Local Setup

Clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/EnderJunk/Work-Week-Loading-Bar.git
cd Work-Week-Loading-Bar
```

No build steps or dependencies required.
